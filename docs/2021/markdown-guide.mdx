Markdown 記法の確認用。md ファイルを汎用的に使いまわせる様に、可能な限り GitHub Flavored Markdown に準拠したい。

```diff javascript
const hello = null;
- console.log("hoge")
+ console.log("hello")
console.log("normal")
```

```js:index.js
function hello (){
  console.log('hello')
}
```

## 見出し heading

```md
# 見出し h1

## 見出し h2

### 見出し h3

#### 見出し h4
```

## リスト

```md
- Hello!
- Hola!
  - Bonjour!
  * Hi!
```

- Hello!
- Hola!
  - Bonjour!
  - Hi!

### 番号付きリスト

```md
1. First
2. Second
```

1. First
2. Second

## テキストリンク

